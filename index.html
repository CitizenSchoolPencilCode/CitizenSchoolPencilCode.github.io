Welcome to Citizen School's Pencil Code Apprenticeship!

In this apprenticeship students learned and applied basic programming concepts. Google's Pencil Code was used to help present these concepts in an easy and fun way. Additionally we took a field trip to Facebook where the students were able to see what an average day is like for software developers.
